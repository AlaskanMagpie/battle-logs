Workflow

  1. Load terrain (GLB preferred, OBJ supported).
  2. Choose what you are placing; left-click on the mesh.
  3. Use Scatter on mesh for random Mana nodes on the surface.
  4. Copy your GLB into public/terrain/ and set Terrain URL to match (e.g. /terrain/myarena.glb).
  5. Export — save as public/map.local.json to override the shipped map (see loadMap.ts).

Green = taps · blue = player relays · red = enemy relays · tall spheres = wizard spawns.
